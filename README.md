# Fire Tournament Tracker

A modern web application for managing and participating in Free Fire tournaments. This platform allows users to browse, join, and track various gaming tournaments with an intuitive interface.

## Features

- Browse available tournaments with detailed information
- Real-time countdown timers for upcoming tournaments
- Secure payment integration for tournament registration
- User profile management
- Admin panel for tournament management
- Responsive design for all devices
- Dark/Light theme support

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Local Storage for data persistence
- Responsive Design
- Modern UI/UX

## Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/FireTournamentTracker.git
```

2. Open `index.html` in your web browser

## Usage

- Browse tournaments on the home page
- Use the search and filter options to find specific tournaments
- Click "Join Tournament" to register for a tournament
- Complete the payment process to confirm your registration
- Access your profile to view registered tournaments
